# lazyutils
command line utility tools for lazy peapoles

## Installation
```sh
git clone "https://github.com/lazypwny751/lazyutils.git"
cd "lazyutils"
make
```

## Usage

### true
```sh
true # it returns 0 stout
```

### false
```sh
false # it returns 1 sterr
```

### mkdir
```sh
mkdir dir1 dir2 dir3
mkdir -p dir1/dir2/dir3
```
